# notebooks/

Lofter 数据研究项目的核心工作目录，按职能分为三个子目录：算法建模、数据分析、数据工程。

每个子目录下按项目独立组织，包含需求文档、技术方案、代码实现和产出物。

## 已有子目录

| 目录 | 简介 |
|------|------|
| `alg/` | 算法模型项目 — 因果推断、推荐算法、用户建模等，侧重**模型设计与训练** |
| `analysis/` | 数据分析项目 — AB 实验分析、用户行为分析、指标归因等，侧重**数据洞察与结论** |
| `engineer/` | 数据工程项目 — ETL 管道、特征工程、数据表构建等，侧重**数据加工与管道** |
